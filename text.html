screenshot text and create PNG

convert PNG to SVG with Inkscape

the hollow effect was a happy accident using 'Edge Detection' in Inkscape's 'Trace Bitmap' tool and I'm keeping it.
Then I turned it into a font with FontForge.
arial-black-hollow.svg arial-black-hollow.ttf arial-black-hollow.woff Arial-Black-Hollow in FontForge format
It's not a finished font as I didn't bother kerning individual letter pairs like 'To' or 'Av', and it's only the most basic ASCII characters but it'll suffice for this demo. I then converted the .ttf font with Hiero creating a font that will work with A-frame, Mozilla's VR platform.
