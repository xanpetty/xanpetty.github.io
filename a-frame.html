A-Frame demo
A-Frame is a Mozilla project. While the examples below are viewable in most current browsers the results currently render best in Firefox.

Continuing from the text demo, Hiero generated both the .fnt and .png files at the same time.
aframe-arial-black-hollow.fnt aframe-arial-black-hollow.png
The font works perfectly with A-frame however it isn't 3D and the letters disappear when the front surface isn't facing the viewer.
There are more efficient ways to display 3D text but as a proof of concept I used Blender to convert an SVG of the letter 'a' from my Arial-Black-Hollow font into a gltf and obj 3D models that can be dragged by a mouse.
a.gltf model binary data for gltf model a.obj model binary data for obj model
The models appear to be identical in A-Frame but you're welcome to compare them by pressing the buttons below.
The aframe-text-geometry-component makes it possible to use 3D text with A-frame.
